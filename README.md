# meta-ad-performance-powerbi-dashboard
This is a Meta Ad Performancr Power BI dashboard that analyzes the effectiveness of ad campaigns across key KPIs such as impressions, clicks, engagements, conversions and budget. It provides a complete funnel views from awareness to engagement to purchases along with demographic, geographic and time-based insights.
